# DeepConnect

Flash cards de connexion progressive — 5 sources, 3 niveaux de profondeur.

## Sources
- **36 Questions · Arthur Aron** — La méthode scientifique originale
- **We're Not Really Strangers** — Format carte, accessible et naturel
- **Where Should We Begin · Esther Perel** — Approche narrative et profonde
- **The And** — Brut, minimaliste, sans filtre
- **Let's Get Deep** — Accessible, flirty, parfait pour un premier date

## Niveaux
- **Surface** — Questions légères, brise-glace
- **Profond** — Connexion réelle, émotions
- **Vulnérable** — Intimité authentique

## Déploiement GitHub Pages

### 1. Créer le repo
```bash
git init
git add .
git commit -m "init: DeepConnect app"
git branch -M main
git remote add origin https://github.com/TON_USERNAME/deepconnect.git
git push -u origin main
```

### 2. Activer GitHub Pages
- Aller dans **Settings → Pages**
- Source : **GitHub Actions**
- Le workflow se déclenche automatiquement à chaque push sur `main`

### 3. Accéder à l'app
```
https://TON_USERNAME.github.io/deepconnect/
```
